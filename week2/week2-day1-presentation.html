What is a CSV File? üìÑ

CSV = Comma-Separated Values

A simple way to store data in a text file

Player,Team,Points,Rebounds,Assists LeBron James,Lakers,25.7,7.3,7.3 Steph Curry,Warriors,29.4,5.1,6.3 Kevin Durant,Suns,28.2,6.7,5.0

First line = column names

Each line after = one player

Commas separate the stats
