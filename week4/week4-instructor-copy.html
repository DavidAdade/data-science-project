Option A: Player Comparison Report üèÄ

Goal: Compare 3-5 professional athletes across multiple statistics and determine who performs best

Deliverables:

  • Comparison table showing key stats for each player
  • At least 2 visualizations (bar charts, scatter plots)

Option B: Team Performance Analysis üèÜ

Goal: Analyze what factors predict whether a team wins or loses games

Deliverables:

  • Team statistics vs. league average comparison
  • Identification of team strengths and weaknesses
  • 2+ visualizations showing team performance
  • Recommendations for how the team could improve

Example Questions:

  • "What makes the Lakers successful this season?"
  • "Why are the [Team] struggling?"
  • "Which players contribute most to [Team]'s success?"

Good for: Students who follow a specific team closely!

Option C: Personal Performance Tracker üìà

Goal: Use their own athletic data to identify what training approaches improve their game

Deliverables:

  • Personal statistics tracked over 5+ games/practices
  • Analysis of performance trends and improvements
  • Visualizations showing progress over time
  • Action plan for continued improvement

Example Questions:

  • "How has my shooting improved this season?"
  • "What practice activities improve my game performance?"
  • "Am I more consistent now than at the start of the season?"

Good for: Students who track their own stats and want personal insights!

Provide Starting Code Templates (Display on screen):

For ALL Projects - Start Here:

# Import libraries import pandas as pd import matplotlib.pyplot as plt # Load the data data = pd.read_csv("nba_stats_enhanced.csv") # Quick look at what you have print(data.head()) print(data.columns)

Option A Template - Player Comparison:

# Select the players you want to compare players = ['LeBron James', 'Giannis Antetokounmpo', 'Kevin Durant'] my_players = data[data['Player'].isin(players)] # Look at key stats print(my_players[['Player', 'PTS', 'REB', 'AST', 'TS%', 'USG%']]) # Create comparison bar chart plt.figure(figsize=(10, 6)) plt.bar(my_players['Player'], my_players['PTS']) plt.xlabel('Player') plt.ylabel('Points Per Game') plt.title('Scoring Comparison') plt.xticks(rotation=45) plt.show()

Option B Template - Team Analysis:

# Filter for your team my_team = data[data['Team'] == 'LAL'] # Compare to league average league_avg_pts = data['PTS'].mean() team_avg_pts = my_team['PTS'].mean() print(f"League Average: {league_avg_pts:.2f} PPG") print(f"Team Average: {team_avg_pts:.2f} PPG") # Show team's top scorers top_scorers = my_team.sort_values('PTS', ascending=False).head(5) print(top_scorers[['Player', 'PTS', 'REB', 'AST']])

Option C Template - Personal Performance:

# Your personal data (modify these with YOUR stats) games = [1, 2, 3, 4, 5, 6, 7, 8] points = [12, 15, 14, 18, 20, 17, 22, 19] # Create line graph showing improvement plt.figure(figsize=(10, 6)) plt.plot(games, points, marker='o', linewidth=2, markersize=8) plt.xlabel('Game Number') plt.ylabel('Points Scored') plt.title('My Scoring Progress This Season') plt.grid(True, alpha=0.3) plt.show() # Calculate average improvement first_half_avg = sum(points[:4]) / 4 second_half_avg = sum(points[4:]) / 4 improvement = second_half_avg - first_half_avg print(f"Improvement: {improvement:.2f} points per game")
Project these templates on screen and have them available as a shared document students can copy from. This removes the "blank page" anxiety.
Activity 4: Independent Work Time
45 minutes

Student Work Session:

Students work independently or in pairs on their projects. They should accomplish:

Minimum Goals for Today:
  • ‚úÖ Load and filter data successfully
  • ‚úÖ Create at least 1 working visualization
  • ‚úÖ Calculate any needed metrics
  • ‚úÖ Write down 2-3 observations about the data

Instructor Role - Circulate and Support:

Walk around the room helping students. Common issues to watch for:

  • Can't filter data: Check spelling of player/team names (case-sensitive!)
  • Charts don't show: Make sure they have plt.show() at the end
  • Too ambitious: Help them simplify if overwhelmed
  • Not sure what to do: Refer them back to templates
  • Finished early: Challenge them to add a second chart or try correlation
Set a timer for every 15 minutes. When it goes off, say "Quick check: raise your hand if you have a chart showing!" This helps you identify who needs help.

Troubleshooting Station:

Have a dedicated area where students can come for one-on-one help. Keep a list of common errors:

  • "KeyError": Column name is misspelled
  • "NameError": Variable doesn't exist (typo or didn't run previous cell)
  • "FileNotFoundError": CSV file not in the right location
  • Empty chart: Data was filtered out completely

üìã Assessment & Differentiation

Project Assessment Criteria:

What You're Looking For:

Technical Skills (40%):

  • Successfully loads and filters data
  • Creates appropriate visualizations
  • Code runs without major errors
  • Uses techniques learned in class

Analysis Quality (40%):

  • Clear research question
  • Appropriate data and metrics chosen
  • Insights are meaningful and well-explained
  • Conclusion answers the research question

Presentation (20%):

  • Clear communication
  • Good pacing and organization
  • Confident delivery
  • Responds well to questions

Differentiation Strategies:

For Struggling Students:

  • Provide more scaffolded templates with fill-in-the-blank sections
  • Reduce requirements to 1 visualization minimum
  • Allow Option C (Personal Tracker) with simpler data
  • Offer one-on-one troubleshooting time
  • Pair with a buddy for support

For Advanced Students:

  • Encourage multiple visualization types
  • Challenge them to use correlation analysis
  • Ask them to compare across multiple seasons
  • Have them mentor struggling classmates
  • Introduce additional advanced metrics

For Students Who Finish Early:

  • Improve visualization aesthetics (colors, labels, titles)
  • Add a third or fourth chart
  • Calculate additional advanced metrics
  • Write a more detailed analysis
  • Create an "executive summary" of findings
