Page 1: Python Basics - What Does This Code Actually Do?

1. Variables - Storing Information

A variable is a name that holds a value. Like a labeled box.
player = "LeBron James"
Stores the text "LeBron James" in a box called player
points = 25
Stores the number 25 in a box called points
fg_pct = 0.52
Stores a decimal number (called a float)
Rules: No spaces in names. Can't start with a number. Case sensitive (pts and PTS are different).

2. Data Types - Kinds of Information

"hello" → String (text, always in quotes)
25 → Integer (whole number)
0.52 → Float (decimal number)
True / False → Boolean (yes or no)
Why it matters:
"5" + "3" → "53" (joins text together)
5 + 3 → 8 (does math)
Quotes make it text. No quotes = a number.

3. print() - Showing Results

print() displays things on screen. Without it, nothing shows up!
print("Hello") → Hello
print(points) → 25
print("Points:", points) → Points: 25
Comma lets you print text and variables together
f-string (fancy way):
print(f"{player} scored {points}")
→ LeBron James scored 25
The f before the quote lets you put variables inside {}

4. Math Operators

+ Add    - Subtract    * Multiply    / Divide
Examples with NBA data:
ppg = total_points / games
Calculate points per game
ppm = pts / minutes
Calculate points per minute
Order of operations: Same as math class!
(a + b) * c → brackets first, then multiply

5. Comparison Operators - Asking Questions

These return True or False. Used for filtering!
> greater than    < less than
>= greater than or equal    <= less than or equal
== equals (is it the same?)    != not equal
Warning: = sets a value. == checks a value.
points = 25 → "set points to 25"
points == 25 → "is points equal to 25?" (True)

6. import - Getting Superpowers

import loads a library (code someone else wrote for you).
import pandas as pd
Loads pandas (data tool). "as pd" is a nickname so you type less.
import matplotlib.pyplot as plt
Loads matplotlib (graphing tool). Nickname: plt
Why nicknames?
Instead of pandas.read_csv() you just type pd.read_csv()
Same thing, less typing!

7. Putting It All Together - Reading Real Code Line by Line

The Code:
import pandas as pd
url = 'https://raw...nba_stats.csv'
df = pd.read_csv(url)
stars = df[df['PTS'] > 20]
print(stars[['Player', 'PTS']])
What It Means (in English):
Load the pandas toolkit
Store the web address of our data file
Read that CSV file, store it in a table called df
Keep only rows where points is more than 20
Show me the Player name and Points columns
Common mistakes:
Forgetting quotes around text: "TOR" not TOR
Using = instead of == when comparing
 
Forgetting parentheses: print() not print
Spelling the column name wrong (copy-paste is your friend!)

Page 2: Pandas & Graphing Cheat Sheet - NBA Data Science

1. Setup - Always Start With This

import pandas as pd
import matplotlib.pyplot as plt
url = 'https://raw.githubusercontent.com/DavidAdade/data-science-project/main/files/nba_stats.csv'
df = pd.read_csv(url)

2. Looking at Your Data

df.head() - first 5 rows
df.head(10) - first 10 rows
df.shape - (rows, columns)
df.columns - list all column names
df.describe() - stats summary
print(df[['Player', 'PTS']]) - show specific columns

3. Filtering Data

One condition:
df[df['PTS'] > 20]
By team name:
df[df['Team'] == 'TOR']
Two conditions (AND = &):
df[(df['PTS'] > 20) & (df['MIN'] > 30)]
Either condition (OR = |):
df[(df['Team'] == 'LAL') | (df['Team'] == 'BOS')]

4. Sorting & New Columns

Sort highest to lowest:
df.sort_values('PTS', ascending=False)
Sort lowest to highest:
df.sort_values('PTS', ascending=True)
Top 10 of sorted:
df.sort_values('PTS', ascending=False).head(10)
Create new column:
df['PPM'] = df['PTS'] / df['MIN']

5. Bar Chart

plt.figure(figsize=(10, 5))
plt.bar(df['Player'], df['PTS'], color='blue')
plt.xlabel('Player')
plt.ylabel('Points')
plt.title('Top Scorers')
plt.xticks(rotation=45, ha='right')
plt.tight_layout()
plt.show()

6. Scatter Plot

plt.figure(figsize=(8, 5))
plt.scatter(df['PTS'], df['AST'],
    alpha=0.5, color='purple')
plt.xlabel('Points')
plt.ylabel('Assists')
plt.title('Points vs Assists')
plt.grid(True, alpha=0.3)
plt.show()

7. Line Graph & Pie Chart

Line graph:
plt.plot(df['Player'], df['PTS'],
  marker='o', color='red', label='PTS')
plt.legend() - shows the labels
Pie chart:
plt.pie(df['PTS'], labels=df['Player'],
  autopct='%1.0f%%')
Shows % of total automatically

8. Quick Reference - Common Patterns

Filter then graph:
top = df[df['MIN'] >= 25]
top = top.sort_values('PTS', ascending=False).head(10)
plt.bar(top['Player'], top['PTS'])
Team averages:
team_avg = df.groupby('Team')['PTS'].mean()
Horizontal bar chart:
plt.barh(team_avg.index, team_avg.values)
Colors: 'blue' 'red' 'green' 'purple' 'orange' 'gold' 'dodgerblue'
Markers (line graph): 'o' circle   's' square   '^' triangle   'D' diamond
