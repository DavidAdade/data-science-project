The Magic of Pandas üêº

Your First Calculated Column

# Load our data
import pandas as pd
data = pd.read_csv("nba_stats.csv")
# Calculate points per minute for EVERYONE!
data['PPM'] = data['PTS'] / data['MIN']
# See the new column
print(data[['Player', 'PTS', 'MIN', 'PPM']].head())

üéâ You just analyzed 450 players in 1 second!

Create Multiple Stats at Once

# Rebounds per game
data['RPG'] = data['REB'] / data['GP']
# Assists per game
data['APG'] = data['AST'] / data['GP']
# Points per shot attempt
data['PTS_per_FGA'] = data['PTS'] / data['FGA']
# See all our new stats
print(data[['Player', 'PPM', 'RPG', 'APG']].head(10))

Advanced Metric: True Shooting %

Better than regular FG% - accounts for 3s and free throws!

# True Shooting Percentage
data['TS%'] = data['PTS'] / (2 * (data['FGA'] + 0.44 * data['FTA']))
# Find the most efficient scorers
top_efficiency = data.sort_values('TS%', ascending=False)
print(top_efficiency[['Player', 'PTS', 'TS%']].head(10))

This is what NBA scouts use! üéØ

Usage Rate: Who Carries the Team?

# Calculate usage rate
data['USG%'] = 100 * ((data['FGA'] + 0.44 * data['FTA'] + data['TOV']) / data['MIN'])
# Find players with highest usage
high_usage = data.sort_values('USG%', ascending=False)
print(high_usage[['Player', 'USG%', 'PTS']].head(10))

High usage = team runs the offense through them

Assist/Turnover Ratio

Great for evaluating point guards!

# Calculate AST/TO ratio
data['AST_TO_Ratio'] = data['AST'] / data['TOV']
# Best decision makers
best_pgs = data.sort_values('AST_TO_Ratio', ascending=False)
print(best_pgs[['Player', 'AST', 'TOV', 'AST_TO_Ratio']].head(10))

Higher = better decision making! üß†

Now Ask Better Questions! üîç

# Find efficient scorers (high PPM AND high TS%)
efficient = data[(data['PPM'] > 0.6) & (data['TS%'] > 0.60)]
print(efficient[['Player', 'PPM', 'TS%', 'PTS']].head())
# Find high-usage players who are still efficient
superstars = data[(data['USG%'] > 25) & (data['TS%'] > 0.55)]
print(superstars[['Player', 'USG%', 'TS%']].head())
# Find undervalued players
# High efficiency but low minutes (could play more!)
undervalued = data[(data['TS%'] > 0.58) & (data['MIN'] < 25)]
