Your First Bar Chart üìä

# Get top 10 scorers
top_scorers = data.sort_values('PTS', ascending=False).head(10)
# Create the bar chart
plt.figure(figsize=(12, 6))
plt.bar(top_scorers['Player'], top_scorers['PTS'], color='blue')
# Add labels and title
plt.xlabel('Player')
plt.ylabel('Points Per Game')
plt.title('Top 10 NBA Scorers')
# Rotate names so they don't overlap
plt.xticks(rotation=45, ha='right')
# Display!
plt.show()

Scatter Plots: Find Relationships üîµ

Shows relationship between TWO variables

Examples:

  • Do players who shoot more score more?
  • More minutes = more points?
  • Assists vs. Turnovers?

Create a Scatter Plot

# Remove missing values
clean = data.dropna(subset=['PTS', 'FG%'])
# Create scatter plot
plt.figure(figsize=(10, 6))
plt.scatter(clean['FG%'], clean['PTS'],
alpha=0.6, s=50, color='purple')
# Labels
plt.xlabel('Field Goal Percentage')
plt.ylabel('Points Per Game')
plt.title('Scoring vs. Shooting Efficiency')
# Add grid for easier reading
plt.grid(True, alpha=0.3)
plt.show()

Reading Scatter Plots üîç

  • Each dot = one player
  • Position shows their stats on both axes
  • Upper-right = high on BOTH stats (best!)
  • Lower-left = low on BOTH stats
  • Look for patterns and relationships

Does better shooting = more points? You tell me! üìä

Compare Teams üè¢

# Calculate average points per team
team_avg = data.groupby('Team')['PTS'].mean()
team_avg = team_avg.sort_values(ascending=False).head(10)
# Horizontal bar chart (easier to read team names)
plt.figure(figsize=(10, 6))
plt.barh(team_avg.index, team_avg.values, color='green')
# Labels
plt.xlabel('Average Points Per Game')
plt.ylabel('Team')
plt.title('Top 10 Highest Scoring Teams')
plt.show()
